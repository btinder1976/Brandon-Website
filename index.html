Build Desk

The decision system.

Every project should move through the same simple path so it does not stay stuck as an idea.

  1. Capture it. Write the idea in one plain sentence.
  2. Pick the outcome. Decide if it should make money, save time, teach, or create memories.
  3. Build version one. Create the smallest useful version.
  4. Test it. Open it, use it, break it, and improve it.
  5. Launch or archive. Ship the useful ones. Park the rest.
